Upspark

The programmable desktop command runner

  • Import Libraries

    The platform belongs to you. Use npm install package-name on Upspark's platform directory to import node libraries. psst, yarn works too

  • Hack and Theme

    There's more to Upspark then just it's logic. You can use CSS to change how you want the runner to look and feel.

  • ES2017 Compliant

    Out the box, use bleeding edge ECMAScript. This includes rest parameters, import statements, fat arrow functions and more.

  • Sensitive Logic

    Need to use passwords or api keys in script logic? Upspark comes with an easy to use AES-256 bit data store.
